# Variables

	- Name rules:

		- Start with a $
		- Followed by letter or underscore
		- Can contain letters, numbers, underscores or dashes
		- No spaces
		- Case-sensitive

	- Examples:

		- Good examples:

			- $item
			- $Item
			- $myVariable
			- $this_variable

		- Bad examples:
		
			- $this-variable
			- $_book
			- $_bookPage

	- Reserved words:

		- http://www.php.net/manual/en/reserved.php
